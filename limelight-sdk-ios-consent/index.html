    Consent Management¶

    The Limelight SDK reads privacy and consent signals from your Consent Management Platform (CMP) and forwards them to every bid request automatically. No additional code is required beyond ensuring your CMP writes the standard IAB UserDefaults keys before the SDK loads ads.

    Supported Frameworks¶

    Framework Source Forwarded in bid request
    TCF 2.0 (GDPR) CMP (UserDefaults) regs.gdpr, regs.ext.gdpr, regs.ext.consent
    CCPA (US Privacy) CMP (UserDefaults) regs.ext.us_privacy
    GPP CMP (UserDefaults) regs.gpp, regs.gpp_sid
    COPPA Programmatic regs.coppa

    TCF 2.0 / GDPR¶

    The SDK reads the standard IAB TCF 2.0 UserDefaults keys written by your CMP. No additional code is required — the SDK picks up these values automatically before each bid request.

    UserDefaults key Description
    IABTCF_gdprApplies Whether GDPR applies ("1" / "0")
    IABTCF_TCString Base64-encoded TCF consent string
    IABTCF_PurposeConsents Purpose-level consent bitstring

    The CMP is the sole source of truth for these values. They are not settable via the SDK API.

    CCPA (US Privacy String)¶

    The SDK reads the CCPA US Privacy string written by your CMP to UserDefaults:

    UserDefaults key Description
    IABUSPrivacy_String IAB US Privacy string (e.g., "1YNN")

    The value is read automatically and forwarded to bid requests as regs.ext.us_privacy.

    COPPA¶

    COPPA is the only consent signal set programmatically. Set it before loading ads for apps directed at children.

    Targeting.shared.subjectToCOPPA = true

    When set to true, the SDK sets regs.coppa = 1 on every bid request.

    GPP (Global Privacy Platform)¶

    The SDK reads the standard IAB GPP keys written by your CMP to UserDefaults:

    UserDefaults key Description
    IABGPP_HDR_GppString Full GPP consent string → regs.gpp
    IABGPP_GppSID Applicable section IDs (underscore-separated, e.g. "2_6_7") → regs.gpp_sid (integer array)

    The SDK parses the underscore-separated SID string written by the CMP into an integer array before forwarding it.

    API Reference¶

    Targeting.shared¶

    Property Type Description
    subjectToCOPPA Bool? Marks the request as subject to COPPA. The only consent signal configurable via the SDK API. Use Targeting.shared — not Limelight — to set this.

    Complete Example¶

    import LimelightSDK

// Called after your CMP has finished and written UserDefaults values.
func onConsentReady() {

    // COPPA — set for apps directed at children
    Targeting.shared.subjectToCOPPA = false

    // Initialize the SDK. TCF, CCPA and GPP signals from UserDefaults
    // are included in every bid request automatically.
    Limelight.initialize(host: "https://your-ad-server.example.com") { result in
        switch result {
        case .success:
            break
        case .failure(let error):
            print("SDK init failed: \(error)")
        }
    }
}
