    LimelightInterstitial (Android)¶

    LimelightInterstitial loads and displays a full-screen interstitial ad. Load the ad in advance, wait for the listener confirmation, then present it.

    Setup¶

    Create an instance with a Context and ad unit ID, set a listener, and call loadAd().

    // 1. Create interstitial
val adUnit = LimelightInterstitial(this, "your-ad-unit-id")

// 2. Configure ad unit
adUnit.setAdSizes(LimelightAdSize(getAdWidth(), getAdHeight()))
adUnit.setInterstitialListener(createListener())

// 3. Load ad
adUnit.loadAd()

    Loading an Ad¶

    Call loadAd() to begin fetching the ad. The SDK notifies you via onAdLoaded() when the ad is ready to show.

    adUnit.loadAd()

    Showing an Ad¶

    To show the ad immediately after the ad is loaded, call show() in the onAdLoaded callback.

    override fun onAdLoaded(interstitial: LimelightInterstitial, ad: LimelightAd) {
    adUnit.show()
}

    Ad Sizes¶

    By default the interstitial occupies the full screen. Use setAdSizes() to set the expected creative size.

    adUnit.setAdSizes(LimelightAdSize(320, 480))
adUnit.loadAd()

    Ad Formats¶

    By default the SDK requests both BANNER and VIDEO formats. Set ad formats before calling loadAd() to restrict or change which formats are requested.

    adUnit.setAdFormats(LimelightAdFormat.VIDEO)
adUnit.loadAd()

    Supported formats: LimelightAdFormat.BANNER, LimelightAdFormat.VIDEO.

    Listener — Ad Lifecycle¶

    Set a listener implementing LimelightInterstitialListener to receive ad lifecycle events.

    Method When it fires
    onAdLoaded(interstitial, ad) Ad loaded and ready to display
    onAdFailedToLoad(interstitial, error) Load error; inspect the LimelightError for details
    onAdImpression(interstitial) Ad became visible to the user
    onAdClicked(interstitial) User tapped the ad
    onAdClosed(interstitial) User dismissed the full-screen ad 
    val listener = object : LimelightInterstitialListener {
    override fun onAdLoaded(interstitial: LimelightInterstitial, ad: LimelightAd) {
        adUnit.show()
    }

    override fun onAdFailedToLoad(interstitial: LimelightInterstitial, error: LimelightError) {
        Log.e(TAG, "Interstitial failed: ${error.message}")
    }

    override fun onAdImpression(interstitial: LimelightInterstitial) {
        // Ad impression tracked
    }

    override fun onAdClicked(interstitial: LimelightInterstitial) {
        // Ad clicked
    }

    override fun onAdClosed(interstitial: LimelightInterstitial) {
        // Ad closed
    }
}

    Public API Reference¶

    Constructor¶

    LimelightInterstitial(context: Context, adUnitId: String)

    Creates an interstitial ad unit for the given ad unit ID.

    Methods¶

    Method Description
    setInterstitialListener(listener) Sets the listener for ad lifecycle events
    setAdSizes(vararg sizes: LimelightAdSize) Sets creative sizes for the ad
    setAdFormats(vararg formats: LimelightAdFormat) Sets creative formats to request
    loadAd() Starts loading the ad
    show() Presents the ad; call only after isReady is true
    destroy() Releases resources; call in onDestroy()

    Complete Example¶

    class InterstitialActivity : AppCompatActivity() {

    private var interstitialAd: LimelightInterstitial? = null

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)

        // 1. Create interstitial
        interstitialAd = LimelightInterstitial(this, "your-ad-unit-id")

        // 2. Configure ad unit
        interstitialAd?.setAdSizes(LimelightAdSize(320, 480))
        interstitialAd?.setInterstitialListener(createListener())

        // 3. Load ad
        interstitialAd?.loadAd()
    }

    override fun onDestroy() {
        super.onDestroy()
        interstitialAd?.destroy()
    }

    private fun createListener(): LimelightInterstitialListener =
        object : LimelightInterstitialListener {
            override fun onAdLoaded(interstitial: LimelightInterstitial, ad: LimelightAd) {
                // 4. Show ad
                interstitialAd?.show()
            }

            override fun onAdImpression(interstitial: LimelightInterstitial) {
                // Ad impression tracked
            }

            override fun onAdFailedToLoad(interstitial: LimelightInterstitial, error: LimelightError) {
                // Handle error
            }

            override fun onAdClicked(interstitial: LimelightInterstitial) {
                // Ad clicked
            }

            override fun onAdClosed(interstitial: LimelightInterstitial) {
                // Ad closed
            }
        }
}
